# MultiStreamer
<img width="64" height="64" alt="image" src="https://github.com/user-attachments/assets/b00f53fb-0d67-4bb6-88d8-390792d74364" />

A clean, modern, OLED-friendly Twitch multistream viewer for Windows. Watch up to 4 streams simultaneously from the people you follow — with chat, without chat, or fullscreen. No ads, no recommendations, no noise.

---

## Features

- **Watch 1–4 streams at once** — streams tile automatically into a clean grid layout
- **Focus Mode** — right click a stream from the followed list to set focus to watch that stream larger than the rest
- **Live following list** — shows all the people you follow who are currently live, sorted by viewer count and refreshed every minute
- **Per-stream chat** — each stream gets its own chat tab, or hide chat entirely
- **OLED-friendly** — Can watch 1-4 streams with that chat hidden and not extra elements on the screen that would cause burn in
- **Fullscreen support** — go fullscreen to hide window elements
- **BTTV & 7TV emote support** — emotes load automatically in chat
- **No nonsense** — no recommendations, no front page, no distractions

## Requirements

- Windows 10 (19041+) or Windows 11
- [WebView2 Runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/) — already included on Windows 11; Windows 10 users may need to install it
- A Twitch account

---

## Installation

Download the latest release from the [Releases](../../releases) page.

---

## Usage

1. Launch the app and log in with your Twitch account
2. The **Live** tab shows everyone you follow who is currently streaming
3. Check up to 4 streamers to add them to the watch view
4. Switch to the **Watch** tab to view your selected streams
5. Use the toolbar to toggle chat or video visibility, or go fullscreen
